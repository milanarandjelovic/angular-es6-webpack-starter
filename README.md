# AngularJS starter kit, Webpack, ES6

AngularJS starter kit, Webpack, ES6.

### Install

###### 1. Clone repo 
`git clone https://github.com/milanarandjelovic/angular-es6-webpack-starter.git`

###### 2. Install dependencies
`npm install` or `yarn`

###### 3. Run the development server
`npm run dev`

###### 4. Run the production
`npm run prod`
